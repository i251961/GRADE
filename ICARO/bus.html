Buscar por URL

A busca é aplicada somente ao campo URL.

campo: url
O limite define quantos registros são buscados (e ficam disponíveis para copiar/exportar) — até 100.000. Na tela sempre aparecem só os primeiros 500. Cada busca cobra moedas proporcionalmente ao retornado. "Verificar disponibilidade" não cobra moedas.
